# react_sanity

## init

```sh
> npx create-react-app .

> npm i -g @sanity/cli

// init studio
> sanity init

// client
> npm i @sanity/client @sanity/image-url @sanity/block-content-to-react
> npm i react-router-dom react-social-icons
```
